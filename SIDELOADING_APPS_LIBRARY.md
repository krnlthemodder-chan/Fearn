## 📦 Installation Instructions

### Using Fearn CLI
```bash
fearn install --app Signal --entitlements default
fearn install --app Telegram --config custom.json
```

### Using Fearn GUI
1. Open Fearn v3.0.0
2. Navigate to App Library
3. Select desired app
4. Click "Install & Sign"
5. Choose certificate and provisioning profile
6. Deploy to device

### Batch Installation
```bash
fearn batch-install --apps Signal,Telegram,Notion --team enterprise
```

## 🔧 Custom Entitlements

Each app can be customized with additional entitlements:

```json
{
  "app": "Signal",
  "base_entitlements": "default",
  "custom_entitlements": {
    "com.apple.developer.siri": true,
    "com.apple.developer.usernotifications.time-sensitive": true
  }
}
```
